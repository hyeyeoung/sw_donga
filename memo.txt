* output.htmld에서 {% %}와 {{}}는 어디에 쓰이는 건지?
https://hleecaster.com/flask-jinja2/
간단 요약: Flask에서는 Jinja2라는 템플릿 엔진을 사용해서 애플리케이션 내 변수와 반복문, 조건문 등을 포함하는 HTML 파일을 렌더링할 수 있다.
render_template()에서 받아온 변수는 {{변수명}}과 같은 식으로 불러와서 쓸 수 있다.
템플릿에서 if 조건문 -> {% if문 %} {% endif %}
템플릿에서 for 반복문 -> {% for문 %} {% endfor %}

아니 근데 윗 내용 output.html에 
<!--
    내용
--> 
이렇게 여러 줄 주석으로 해서 넣으면 왜 오류 나냐?
